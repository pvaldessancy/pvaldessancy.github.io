# Paula Valdés Sancy - Portafolio Profesional Digital

Este proyecto consiste en el desarrollo de un portafolio profesional digital para la artista visual chilena **Paula Valdés Sancy**. La aplicación está diseñada para presentar de manera elegante y minimalista su trayectoria, obras y reconocimientos, proporcionando una plataforma centralizada para su presencia en línea.

## Objetivos del Proyecto

- **Exhibición de Obras**: Presentar una galería interactiva con las obras más destacadas de la artista, incluyendo detalles sobre la técnica, dimensiones e inspiración de cada pieza.
- **Identidad Digital**: Establecer una presencia profesional en la web que refleje la estética y sensibilidad de la artista.
- **Conectividad**: Facilitar el contacto directo con la artista a través de enlaces a redes sociales, correo electrónico y teléfono.
- **Resposividad**: Garantizar una experiencia de usuario óptima en cualquier dispositivo (móvil, tablet, escritorio).
- **Optimización**: Implementar mejores prácticas de SEO y rendimiento para mejorar la visibilidad y velocidad de carga.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del contenido.
- **CSS3 (Vanilla)**: Diseño minimalista, animaciones personalizadas y diseño responsivo sin dependencias externas.
- **JavaScript (Vanilla)**: Lógica para la galería lightbox, efectos de desplazamiento suave y navegación interactiva.
- **Google Fonts**: Tipografías modernas (Outfit y Playfair Display) para una lectura fluida y estética premium.
